# Image Assets

This directory is prepared for local-hosted profile, research, and logo assets.

## Recommended folders
- `members/` → PI/postdoc/student photos
- `research/` → research-theme visuals used in `research.md`
- `logos/` → partner/company logos
- `impact/` → awards, transfer, and impact visuals

## Suggested filenames used by templates/data
- `members/jihoon-kim.jpg`

- `research/biorefinery.jpg`

- `impact/nrf-logo.png`

## Image size guidance (recommended)
- Member photo cards: **900 x 1200 px** (3:4 portrait)
- PI hero/profile photo: **1200 x 1500 px** (4:5 portrait)
- Research theme image: **1600 x 1000 px** (16:10 landscape)
- Logos: transparent PNG, width around **400–800 px**

## File-size guidance (recommended)
- Member photos: under **500 KB** each
- Research images: under **800 KB** each
- Logos/icons: under **300 KB** each
- Keep total `assets/images/` practical for GitHub Pages performance (ideally < 30–50 MB).

## Format guidance
- Photos: `.jpg` (quality ~75–85)
- Graphics/logos requiring transparency: `.png`
- Prefer sRGB color profile.

If external hosting URLs are unavailable, place files above and update `_data/members.yml` / `_data/research.yml` / related markdown pages to local paths.
